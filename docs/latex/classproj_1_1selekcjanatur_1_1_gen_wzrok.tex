\doxysubsection{Opis szczegółowy}
Klasa implementująca gen zasięgu wzroku. 

\doxylink{interfaceproj_1_1selekcjanatur_1_1_gen}{Gen} określa jak daleko człowiek widzi w symulacji. Wartość genu jest z zakresu 1.\+0-\/10.\+0 gdzie\+:
\begin{DoxyItemize}
\item 1.\+0 = minimalny zasięg wzroku (1 komórka)
\item 10.\+0 = maksymalny zasięg wzroku (10 komórek)
\end{DoxyItemize}

Dziedziczenie odbywa się z możliwością mutacji ±1.0, przy czym wartość jest zawsze ograniczana do zakresu 1-\/10. Konstruktor prywatny jest używany tylko przy dziedziczeniu.
