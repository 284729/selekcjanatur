\doxysubsection{Opis szczegółowy}
Klasa implementująca gen prędkości chodzenia. 

\doxylink{interfaceproj_1_1selekcjanatur_1_1_gen}{Gen} określa z jaką częstotliwością człowiek porusza się w symulacji. Wartość genu jest z zakresu 1.\+0-\/3.\+0 gdzie\+:
\begin{DoxyItemize}
\item 1.\+0 = najwolniejszy ruch (co 3 klatki)
\item 2.\+0 = średnia prędkość (co 2 klatki)
\item 3.\+0 = najszybszy ruch (co klatkę)
\end{DoxyItemize}

Dziedziczenie odbywa się z możliwością mutacji ±0.75, przy czym wartość jest zawszę ograniczana do zakresu 1-\/3.
